# 🔐 Password Strength & Breach Checker

A beginner-friendly cybersecurity tool that:
- Checks password strength (length, uppercase, lowercase, numbers, special characters).
- Checks if the password has been exposed in known breaches using the [Have I Been Pwned API](https://haveibeenpwned.com/Passwords).
